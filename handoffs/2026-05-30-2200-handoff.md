## Blocages / échecs

- **Erreur "Erreur lors de l'envoi"** : cause racine = serveur Express lancé à 20:45 avec l'ancien code (avant refonte `quantiteParCafe`). Résolu en killant le process et relançant `npm run dev`.
- **Logo trop lourd** : `Logo principal - fond 3.png` = 6250×6250 px, 308 KB → PDF gonflait à 431 KB. Résolu par remplacement avec `logo-amalivre.jpeg` (58 KB) → PDF 226 KB.
- **Jost jamais chargé** : Jost était référencé dans les composants React mais absent du `index.html` (Inter à la place). Le navigateur tombait en fallback Arial. Corrigé en remplaçant par Montserrat correctement chargé.
- **KALUFONIA / Classy Vogue** : polices non disponibles sur Google Fonts (fonts custom/premium). Substitutions : Cinzel ← KALUFONIA, Cormorant Garamond ← Classy Vogue. Si l'utilisateur fournit les fichiers `.woff2`, les intégrer via `@font-face`.
