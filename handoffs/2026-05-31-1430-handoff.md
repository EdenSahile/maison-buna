**Agent 2 — Développeur** (`developer`)
- Lit : `reports/dev-report.md`

- Écrit : `reports/dev-report.md` → "Développeur — DONE"

**Agent 3 — Testeur** (`tester`)
- Lit : `reports/dev-report.md`
- Fait : vitest run, tests HTTP complets (200/400/415/500), vérif PDF logo, mail test

**Agent 4 — Code Reviewer** (`code-reviewer`)
- Lit : `reports/test-report.md`
